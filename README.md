# Local Docs Hub

Local Docs Hub is a lightweight local Markdown viewer for folders that contain project documentation.

## Features

- connect to any local root folder
- browse Markdown files in a folder tree
- render tables, images, and Mermaid diagrams
- follow relative links between Markdown files
- print the current page to PDF from the browser

## Start

```powershell
npm install
npm start
```

Then open `http://127.0.0.1:4120`.

## Intended Usage

- docs generated by AI in Markdown
- business and technical documentation
- local-first review before publishing elsewhere
